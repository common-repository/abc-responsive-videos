== Description ==
Automatically makes all videos responsive on handheld and mobile devices.

== Installation ==
Upload the ABC Responsive Videos plugin to your blog and activate it.

1, 2, 3: You&rsquo;re done!

== Frequently Asked Questions ==

1) How can I ignore certain videos?

Surround the video with a DIV container that uses the class &rsquo;responsiveignore&rsquo;.

2) Can I set a specific container class or ID for responsive videos?

Yes. Go to the Customizer and click on the Responsive Video section. Enter the class or ID selector you want to use in the appropriate text field. By default, the plugin uses the ID &rsquo;#page&rsquo;.
